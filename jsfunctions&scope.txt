8. Closures

> A closure is a function that remembers and can access the variables of its outer function even after the outer function has finished executing.

Example:

function outer() {
    let count = 0;

    function inner() {
        count++;
        console.log(count);
    }

    return inner;
}

const counter = outer();

counter(); // 1
counter(); // 2
counter(); // 3

9. this Keyword

> The `this` keyword refers to the object that is currently executing the function.
> Its value depends on how the function is called.

> in arrow function this doesnot have own 

Example:

const person = {
    name: "Virat",

    greet() {
        console.log(this.name);
    }
};

person.greet();
